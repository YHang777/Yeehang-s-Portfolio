AR Furniture App (FYP)

Augmented Reality Visualization & Interior Design

Project Overview

This project addresses the common challenge of "visualizing before buying." It is a mobile application that allows users to place high-fidelity 3D models of furniture into their real-world environment using Augmented Reality. The system integrates AI assistance to help users choose furniture and saves layout designs for future reference.

Key Features Developed:

  • Smart Catalog: Cloud-connected inventory system fetching real-time data from Firebase.
  • AR Visualization Engine: Detects floor planes to anchor 3D objects with stability.
  • Design Persistence: Saves AR placement coordinates so users can recover their room layout later.
  • AI Consultant: Integrated chatbot for furniture inquiries with smart cart actions.
System Design & Interface
1. Product Catalog Interface: Users can browse categories (e.g., Chairs, Tables). The UI displays essential details like price and dimensions, with a "View in AR" button to launch the camera.
2. Real-time AR Placement: Using surface detection, the app identifies the floor and places the 3D model. Users can rotate and move the furniture to see exactly how it fits in their space.
3. Saved Designs & Layout Recovery: Users can save their current AR session. This stores the specific placement and orientation of furniture, allowing them to reload a previous room arrangement instantly without needing to redesign from scratch.
4. AI Interior Assistant: An integrated AI module allows users to ask questions about furniture materials or style advice. It features an instant "Add to Cart" button for convenience.
(Future Roadmap: Implementing auto-popups where the AI suggests items based on keywords detected in the conversation.)
