AR Furniture App (FYP)

Augmented Reality Visualization & Interior Design

Project Overview

This project addresses the common challenge of "visualizing before buying." It is a mobile application that allows users to place high-fidelity 3D models of furniture into their real-world environment using Augmented Reality. The system ensures that the virtual furniture scales correctly to the room, helping users make informed purchasing decisions.

Key Features Developed:

  • Smart Catalog: A cloud-connected inventory system fetching real-time data and 3D assets from Firebase.
  • AR Visualization Engine: Built with ARCore to detect floor planes and anchor 3D objects with stability.
  • Model Management (Python): A backend script that optimizes and processes raw 3D model files for mobile performance before storage.
System Design & Interface
1. Product Catalog Interface: Users can browse categories (e.g., Chairs, Tables). The UI displays essential details like price and dimensions, with a "View in AR" button to launch the camera.
2. Real-time AR Placement: Using surface detection, the app identifies the floor and places the 3D model. Users can rotate and move the furniture to see exactly how it fits in their space.
3. Admin & Backend Processing: The admin side utilizes Python scripts to handle 3D model uploads. It ensures models are optimized for mobile rendering and updates the Firebase Firestore database seamlessly.
