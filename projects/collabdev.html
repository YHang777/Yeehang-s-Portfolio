SafeHaven

Mental Health Support Platform & Therapy System

Project Overview

SafeHaven is a comprehensive platform designed to provide individuals with access to professional guidance, self-help resources, and a supportive community. The system enables users to connect with mental health professionals for confidential one-on-one video therapy sessions.

Key Modules Developed:

  • Professional Therapy Help: Video conferencing, appointment scheduling, and therapist selection.
  • Song Recommendation Engine: Analyzes user mood (Happy, Sad, Neutral) to fetch personalized playlists using the YouTube API.
  • Community Forum: A safe space for users to share experiences, featuring automated moderation to filter inappropriate content.
System Design & Interface
Song Recommendation Module: Users select their current mood, and the system utilizes the YouTube API to recommend curated playlists (e.g., Lo-Fi, Relaxing Music) to aid in emotional regulation.
